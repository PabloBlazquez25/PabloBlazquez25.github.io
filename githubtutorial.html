Como crear una página WEB personal desde GitHub:

Desde ubuntu, si no tenemos instalado git deberemos hacerlo, para esto escribiremos en la consola: $ sudo apt install git init

A continuación iniciaremos git: $ git init
Escribimos el origen de nuestra página web: $ git remote add origin https://github.com/PabloBlazquez25/PabloBlazquez25.github.io
Lo subiremos al servidor: $ git pull origin master
Añadiremos los archivos que deseamos mostrar a github: $ git add css docker.html index.html index.html js snd img
Actualizacmos: $ git commit -m "Primer commit"
Y por último lo lanzamos al servidor: $ git push origin master (Usuario y contraseña)
